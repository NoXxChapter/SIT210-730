Code:

Setup -> With each line tell which PIN to send/receive signal
Loop: 
	If (Switch == ON) -> execute:
		porchLED -> ON		hallLED -> ON
		wait 30sec
		porchLED -> OFF
		wait 30sec
		hallLED -> OFF
Last line: to help easier to observe the actuator.

Each part:

	+ The NANO IOT33 act as the central processor (control processor) it contains and execute command (code) embedded in the memory to do the task.
	+ LED: act as light source.
	+ Pipes: connect PIN, transit electricity.
	+ Switch: act as a toggle decide when to allow the transit.
	+ Resistor: limit the voltage, protect LED from being damaged.

	===============================================================================================

How it work ?

First with switch, i connect the middle leg with PIN12 to send the switch signal, with both left and right legs are connected to 3.3V and Ground. After received the signal from switch, PIN11 and PIN10 then send HIGH signal to make each LED light up. With porch LED(LED11) stays for 30 secs and hall LED(LED10) stays for 60sec. Then, turn off.
